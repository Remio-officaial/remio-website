# CLAUDE.md

이 파일은 Claude Code가 이 프로젝트에서 작업할 때 참고하는 지침서입니다.

## 프로젝트 개요

- **프로젝트명**: 레미오 (REMIO)
- **설명**: 홍대입구역 2번 출구 도보 3분 거리의 레트로 미니 오락실 웹사이트
- **운영 서비스**: 인형뽑기, 가챠머신
- **판매 상품**:
  - 12지신 액막이 키링 12종 (15,000원) - 자체 제작
  - 오행음양 팔찌 5종 (25,000원) - 자체 제작
- **주요 기능**:
  - 무료 사주오행 분석
  - 카카오톡 공유 후 순금 1돈 이벤트

## 기술 스택

- 순수 HTML / CSS / JavaScript
- 단일 파일 구조 (`remio-final.html`)
- 외부 프레임워크 없음
